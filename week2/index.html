Chapter 2: Programming Basics

  • remember to always place comments in JS so you know what the code is supposed to do
  • Javascript language is a lot like C Programming language (yep!!!)
  • JS is made of statements that end with a ;
  • Block: a series of statements that are collected together inside {}
  • Whitespace: spaces, tabs, new lines that are blank. Helps make reading code easier
  • Primitive Data Types: string, symbol, number, boolean, undefined, null
  • Object Data Types: arrays, functions, object literals
  • Operand: when an operator applies an operation to a key ex. typeof 'hello'; operator is typeof and operand is the string 'hello'
  • Variables: used in programming languages to refer to a value stored in memory but must be declared; in JS we use const or let (not usually var) to declare the variable and then name the variable whatever we want (a const variable value never changes, a let variable value can change)
  • you can declare and assign multiple variables at the same time if you seperate them with a ,
  • const variable values can be changed if they are object Types ex. const name = {value: 'Alexa'}; name.value = 'Siri' changed the name value from Alexa to Siri
  • const and let only work in the block they are declared in, a local scope is inside a block or function, a global scope can be used within the whole page (not declared in a block)
  • global variables can be overwritten within a block, ex. let b = 2; {let b = 4; b;} the outcome is 4 when printing the b inside a block and 2 outside the block
  • if you assign a variable within the block but don't decare it, the value has a global scope
  • use camel case to name variables ex. firstNameAndLastName
  • changing the value of referenced object changes the objects value too, ex. const c = { value: 1}; let d = c; (referenced c.value=1 and d.value=1); d.value = 2 (d.value=2 and c.value=2)
  • Properties: info about the object or value
  • Methods: perform an action on the object or value, end with parenthesis
  • you can use + to connect strings or just use teplate literals (I would use template literals, easier; ``
  • Template Literal: `` to connect all kinds of strings instead of '' + '' ex. const name = `Siri`; `Hello ${name}!`; outcome: 'Hello Siri!'
  • **What are symbols and why do we use them/need them?
  • numbers, whether they are integers (whole numbers) or floats (decimal) all are the same in JS
  • 0x numbers: hexadecimal numbers often used for color codes, ex. 0xAF (A=10, F=15)
  • 0o numbers: octal literals ex.0o47 (4 eights and 7 units
  • 0b numbers: binary notation ex. 0b1010 (1 eith, 0 fours, 1 two and 0 units)
  • Exponential numbers: JS will do these ex.1e6 (1 * 10 ^ 6) = 1000000) or 2E3 (2 * 10^3) = 2000
  • Decimal numbers: created by using a negative index value ex. 2.5e-3 (2.5 * 10^-3) = 0.0252
  • I didn't know you could use compound assignment operators for subtraction, mulitplication, division, and remainders too
  • Infinity is used to represent a number that is too big for JS, -Infinity for a number too small
  • Nan error: not a number
  • Type Coercion is JS trying to put two different types together, best not to rely on this ex. '8' + 2; outpout:'28'
  • Convert strings to numbers by using the Number method ex. Number('23'); output 23
  • Convert numbers to strings by using the String method ex. String(3); ouptut: '3'
  • Truthy values: boolean values that come out true
  • Falsy values: boolean values that come out false (usually empty strings, empty template literals, 0, -0, NaN, false, null, undefined
  • Strings are case-sensitive, so upper-case letters are considered to be less than lower-case letters

Chapter 3: Arrays, Logic, and Loops

  • Array: ordered list of values
  • Index: the position of a varaible in an array using square brackets ex. const heroes = []; heroes[0];
  • Array Literals: an array (list) with values already inside
  • Deleting a value in an array doesn't delete its space, the array is still the same length, that spot just becomes undefined
  • Destructuring an Array: concept of taking values out of an array and presenting them as individual values
