Object Methods: this

  • Method: functions that are stored in object properties
  • OOP: Object-Oriented Programming
  • To access the object, a method can use the "this" keyword
  • Use "this" instead of just referencing the outer variable (name of the object) is to avoid errors
  • "This" is "free": can be used for multiple different objects when a function is called. By using "this", you can use functions like they are meant to be used, not just for one object at a time, but many.
  • Arrow functions don't have their own "this"; they take info from the outer "normal" function
  • The value of "this" is defined at run time
  • Ladder object allows you to do up and down with numbers

This in JavaScript

  • "this" in a global context defaults to the window ojbect, but this isn't usually used
  • "this" in an ojbect construction refers to the instance, but only when calling the function with "new" (ex. function Person(age) { this.age = age; } const greg = new Person(22);
  • Use "bind" to use "this" to reference the object in an event listener
  • "bind" doesn't work with arrow functions
  • "this" is a crucial keyword in JavaScript
